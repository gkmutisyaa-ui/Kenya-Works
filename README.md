# KenyaWorks

Kenya's gig economy platform — connecting skilled workers with employers across Nairobi. Payments via M-Pesa.

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Payments**: M-Pesa Daraja API
- **Hosting**: Vercel

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → auto-deploys via Vercel
